# FAQ for SQL DB in Fabric Hands-On Workshop
**FabCon 2026**


## 1. Sign in to Microsoft Fabric
Sign in to **Microsoft Fabric** using the **Azure Active Directory (AAD)** credentials provided by the **Speakers/Proctors**.

---

## 2. Update Temporary Password
The password provided for this lab is a **temporary password**.  
Please change it immediately to a password you can easily remember or securely note down for future use.

---

## 3. Complete App Authentication
After signing in, you will be redirected to the **application authentication flow**.  
Complete the authentication setup before starting the hands-on exercises.

---

## 4. Use Private/Incognito Browser Mode
As a best practice, perform all workshop modules using a **private/incognito browser window** to avoid session or caching issues.

---

## 5. GitHub Sign-in for VS Code
In **Visual Studio Code**, sign in to **GitHub** using your **personal GitHub account** to take advantage of **GitHub Copilot features and agents**.

---
